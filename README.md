# Gym OOP Project

This project is a small Java gym management system.

## Build

Use Maven to build and manage dependencies:

```bash
mvn clean compile
```

## Run

Add your own application entry point or extend `gym.DataManager` for JSON persistence.

## Notes

- Source files are located under `src/main/java/gym`
- Jackson is configured via `pom.xml` for JSON binding and Java 8+ date support
